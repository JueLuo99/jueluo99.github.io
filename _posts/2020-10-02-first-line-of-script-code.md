脚本语言共同遵守的规则：当第一行为 ```#!/path/to/script/interpreter``` 时，指定了用来执行本脚本的解释器

例如：

| 解释器路径 | 脚本语言 |
|---|---|
| ```#!/bin/sh``` | shell 脚本 |
| ```#!/usr/bin/perl``` | perl 脚本 |
| ```#!/usr/bin/python``` | python 脚本 |
| ```#!/usr/bin/python3``` | python3 脚本 |
| ```#!/usr/bin/python2``` | python2 脚本 |
