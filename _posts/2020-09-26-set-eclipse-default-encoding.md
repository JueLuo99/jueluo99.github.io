在 Eclipse 2020-03 版本中，发现默认保存文件使用的编码格式为 GBK

将其修改为 UTF-8 的步骤如下：

1. 依次打开 ```Windows``` -> ```Preferences```
1. 找到 ```General``` -> ```Workspace``` -> 右侧选项界面中的左下角 ```Text file encoding``` -> 选择所需的编码格式
