## 代码块

行内代码块：

一个 C 程序总是从 `main()` 函数开始执行的

不指定语言的代码块：

```
#include <stdio.h>

int main()
{
   printf("Hello, World!");
   return 0;
}
```

指定语言的代码块：
```c
#include <stdio.h>

int main()
{
   printf("Hello, World!");
   return 0;
}
```
